# tolvi-solo

Vault schema packs for solo builders. This repo ships the Engineer pack today; Writer, Entrepreneur, and CPA packs are planned.

## Structure

```
packs/<name>/          — schema pack (templates, README)
hooks/                 — Claude Code session hooks (tolvi-recall, tolvi-sync)
install.sh             — provisions vault/ in any git repo
vault/                 — this repo's own vault (tracked, engineer pack)
```

## Vault

Decisions and sessions about tolvi-solo itself live in `vault/`. Use the templates in `vault/templates/` as starting points.
